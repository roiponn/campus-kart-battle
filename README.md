# Campus Kart Battle

夜のキャンパス祭を舞台にした、単一HTMLの3Dカートレースゲームです。

## Controls

- `W` / Arrow Up: 加速
- `S` / Arrow Down: ブレーキ / 後退
- `A` / `D` / Arrow Left / Arrow Right: ステアリング
- `Space`: ドリフト
- `E`: アイテム
- `M`: サウンド切り替え

## Run Locally

```bash
python3 -m http.server 55083
```

Then open `http://localhost:55083/`.
